提权相关信息

limit 可接收一个或两个整型参数

  1. 接收一个参数

    SELECT * FROM table LIMIT 5; //检索前5个记录行 换句话说，LIMIT n 等价于 LIMIT 0,n。

  2. 接收两个参数（第一位表示从第几行开始偏移，第二位表示偏移最大数目）

    SELECT * FROM table LIMIT 5,10; // 检索记录行 6-15
